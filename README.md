# 寻道大千联盟信息管理系统

一个适合毕业设计展示和答辩的完整本地项目，包含前台门户、后台管理、管理员登录、SQLite 数据库与联盟成员信息维护。

## 功能

- 联盟主页总览
- 妖盟大全与成员资料维护
- 联盟战力排行榜
- 联盟公告与瓜棚动态
- 管理员登录
- 成员新增、编辑、删除
- 公告新增、编辑、删除
- SQLite 本地数据库持久化

## 项目结构

- `server.py`: Python 后端服务
- `public/index.html`: 前端页面
- `public/styles.css`: 页面样式
- `public/app.js`: 前端交互逻辑
- `data/alliance.db`: 程序首次运行后自动生成的数据库文件

## 启动方式

在项目目录运行：

```powershell
python server.py
```

然后浏览器打开：

```text
http://127.0.0.1:8000
```

- 密码：`123456`
